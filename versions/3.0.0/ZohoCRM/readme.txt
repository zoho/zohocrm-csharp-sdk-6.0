Zoho CRM C# SDK 3.0.0
————————————————————————

This is the readme file for Zoho CRM’s C# SDK version 3.0.0.

This file gives a brief of the enhancements and/or bug fixes in the latest version.

----------------
Enhancements
----------------
- Changed favorite field datatype in CustomViews.
- Support for the following new APIs.
    - CustomViews
        - Change Sort Order of Custom Views (https://www.zoho.com/crm/developer/docs/api/v6/sort-order-cv.html)
    - Users
        - Users Count (https://www.zoho.com/crm/developer/docs/api/v6/users-count.html)
    - Features
        - User Licences Count (https://www.zoho.com/crm/developer/docs/api/v6/get-user-licences.html)

You can also take a look at our GitHub page here (https://github.com/zoho/zohocrm-csharp-sdk-6.0/blob/master/README.md)
